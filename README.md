# 🐱 WeatherCat

WeatherCat is a simple Python project that shows the weather of a city and the mood of a cat based on weather conditions.

## Features

- Fetches real-time weather data using the OpenWeather API
- Displays temperature, humidity and weather conditions
- Shows a cat mood based on the weather

## Technologies

- Python
- Requests
- OpenWeather API

## Example

Weather in London

Temperature: 12°C  
Condition: light rain  
Humidity: 81%

Cat mood: 🌧️ Watching the rain
